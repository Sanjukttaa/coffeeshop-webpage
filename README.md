# ☕ Coffee Shop Webpage

A modern and aesthetic coffee shop website built using HTML, CSS, and JavaScript.
This project showcases a stylish café experience with multiple sections including menu items, gallery, about page, and contact section.

## ✨ Features

* ☕ Attractive coffee shop landing page
* 🍔 Food and beverage showcase
* 🖼️ Image gallery section
* 📄 About Us page
* 📞 Contact section
* 🎨 Responsive and visually appealing UI
* ⚡ Smooth frontend interactions using JavaScript

## 🚀 Tech Stack

* HTML
* CSS
* JavaScript

## 📂 Project Setup

1. Clone the repository

```bash
git clone https://github.com/Sanjukttaa/coffeeshop-webpage.git
```

2. Open the project folder

```bash
cd coffeeshop-webpage
```

3. Run the project

Open the `star.html` file in your browser.
